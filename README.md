📝 Todo File Manager (Python)
📌 Overview

This is a simple Python project that helps you:

Create a to-do file
Add tasks to it
Rename the file
Delete the file if needed

It is a beginner-friendly project to learn file handling in Python.

🚀 Features
Create or reset a todo file
Add multiple tasks interactively
Rename file safely
Delete file with confirmation
Error handling (file not found, permission issues)
▶️ How to Run
Make sure Python is installed
Run the script:
python todo_manager.py
🧠 What You Learn
File handling (open, write, append)
OS module (os.rename, os.remove)
Loops and user input
Error handling (try/except)
Basic program structure
